## Getting Started

### Accessing NeoServe
1. Navigate to your NeoServe instance URL
2. Select your user type from the landing page:
   - **Employee Portal**: For team members and support staff
   - **Admin Portal**: For managers and administrators
   - **Customer Support**: For customers seeking help

### First Time Login
- Use the demo credentials provided or your assigned login details
- Complete your profile setup after first login
- Enable browser notifications for real-time updates

## Employee Portal

### Dashboard Overview
The employee dashboard provides:
- **AI Chat Assistant**: Get instant help and guidance
- **Mood Tracker**: Log your daily mood and wellbeing
- **Ticket Management**: View and manage assigned tickets
- **Performance Metrics**: Track your productivity and achievements

### Using the AI Assistant
1. Type your question in the chat interface
2. The AI will provide contextual responses based on your role
3. Use suggested responses for common queries
4. Chat history is saved for reference

### Mood Tracking
1. Use the emoji slider to rate your current mood (1-10)
2. Add optional notes about your day
3. View your mood trends over time
4. Managers can see aggregated team mood data

### Managing Tickets
- **View Active Tickets**: See all tickets assigned to you
- **Update Status**: Change ticket status as you work
- **Add Comments**: Communicate with customers and team
- **Set Priority**: Adjust priority based on urgency

## Troubleshooting

### Common Issues

#### Login Problems
- **Forgot Password**: Use the password reset link
- **Account Locked**: Contact your administrator
- **Browser Issues**: Clear cache and cookies
- **Network Problems**: Check internet connection

#### Chat Assistant Not Responding
- **Refresh Page**: Reload the browser tab
- **Check Connection**: Ensure stable internet
- **Clear Browser Data**: Reset local storage
- **Contact Support**: Report persistent issues

#### Notifications Not Working
- **Browser Permissions**: Enable notifications in browser settings
- **Do Not Disturb**: Check system notification settings
- **Extension Conflicts**: Disable ad blockers temporarily
- **Update Browser**: Use latest browser version

#### Performance Issues
- **Slow Loading**: Check internet speed and server status
- **Memory Usage**: Close unnecessary browser tabs
- **Cache Issues**: Clear browser cache and reload
- **System Resources**: Ensure adequate device memory
